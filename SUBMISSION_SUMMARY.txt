API ENDPOINTS (25+)
────────────────────────────────────────────────────────────────

Authentication:
  POST   /api/auth/register
  POST   /api/auth/login
  GET    /api/auth/user

Events:
  GET    /api/events (with filters)
  GET    /api/events/:id
  POST   /api/events (admin)
  PUT    /api/events/:id (admin)
  DELETE /api/events/:id (admin)
  POST   /api/events/check-availability

Bookings:
  POST   /api/bookings
  GET    /api/bookings
  GET    /api/bookings/:id
  PUT    /api/bookings/:id/cancel

Payments:
  POST   /api/payments/create-intent
  POST   /api/payments/process
  GET    /api/payments/history

Admin:
  GET    /api/admin/dashboard/stats
  GET    /api/admin/bookings
  GET    /api/admin/users
  PUT    /api/admin/users/:userId/role
  DELETE /api/admin/users/:userId

DATABASE SCHEMA
────────────────────────────────────────────────────────────────

Collections:
  • User (accounts, authentication)
  • Event (event listings, details)
  • Booking (ticket reservations)
  • Payment (transaction records)
  • Notification (alerts, reminders)

Relationships:
  • Event → User (organizer)
  • Booking → User (customer)
  • Booking → Event (booked event)
  • Payment → Booking
  • Notification → User

QUICK START
────────────────────────────────────────────────────────────────

1. Backend Setup:
   cd backend
   npm install
   cp .env.example .env
   npm run dev

2. Frontend Setup (new terminal):
   cd frontend
   npm install
   npm start

3. Access Application:
   Frontend: http://localhost:3000
   Backend API: http://localhost:5000
   MongoDB: mongodb://localhost:27017
