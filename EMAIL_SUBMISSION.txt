💾 DATABASE SCHEMA
────────────────────────────────────────────────────────────────────────────────

User Collection:

- createdAt, updatedAt

Event Collection:
- title, description, category, location, date, startTime, endTime
- image, price, totalSeats, availableSeats, organizer (ref)
- createdAt, updatedAt

Booking Collection:
- bookingId (unique), event (ref), user (ref), numberOfTickets
- totalPrice, status (pending/confirmed/cancelled)
- paymentStatus (pending/completed/failed)
- bookingDate, notes

Payment Collection:
- booking (ref), user (ref), amount, currency
- paymentMethod (stripe/paypal), stripePaymentIntentId
- status (pending/succeeded/failed), transactionId (unique)
- paymentDate, createdAt

Notification Collection:
- user (ref), type, title, message
- relatedBooking (ref), relatedEvent (ref)
- isRead, createdAt
